# DRXP Store

Fashion ecommerce project.

---

# Stack

Next.js
TypeScript
Tailwind
Prisma
PostgreSQL
Zustand

---

# Install

npm install

---

# Environment

Создать .env файл.

DATABASE_URL=

---

# Database

npx prisma generate

npx prisma migrate dev

---

# Seed data

npx prisma db seed

---

# Run project

npm run dev
